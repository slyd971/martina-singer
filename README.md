# Martina Singer Press Kit

Press kit premium Next.js 16 pour Martina.

## Local development

```bash
npm run dev
```

Puis ouvrir `http://localhost:3000`.

## Structure

- `src/data/artists`: donnees locales typees
- `src/lib/seo`: metadata, sitemap et robots
- `src/components/press-kit.tsx`: presentation premium reutilisable
- `src/app/(site)`: pages shared App Router

La version actuelle est configuree uniquement pour Martina.
